plugin for CudaText.
reads selected math expression, e.g. "2.4*sin(pi/3)" and 
replaces selection with number result.
math functions from Python supported:

abs
acos
asin
atan
atan2
ceil
cos
cosh
degrees
e
exp
fabs
floor
fmod
frexp
hypot
ldexp
log
log10
modf
pi
pow
radians
sin
sinh
sqrt
tan
tanh

details: https://docs.python.org/3/library/math.html
